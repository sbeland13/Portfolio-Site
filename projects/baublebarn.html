Project Details

  • Nov 2015
  • AngularJS, Node.js, AWS (S3), Digital Marketing
  • Live Site (coming soon)

This E-commerce site was originally put together on the Shopify platform, but was later taken down and is being rebuilt from the ground-up using AngularJS, complete with secure OAuth login for users to log-in to their shopping cart. The application is built using the Moltin.io API to handle the inventory and payment integrations. This is an ongoing project with more features being added over time.
