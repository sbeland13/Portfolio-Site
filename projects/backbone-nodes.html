Project Details

  • Mar 2016
  • BackboneJS, D3, Firebase, Socket.io
  • Live Site

This (relatively useless) sample application was built to showcase how to easy it is to integrate the BackboneJS framework with a backend API like Firebase. Each node and each connecting segment has its own entry into the database. A node can be given a string, as well as a text color for that string. The code for the SVG force rendering was inspired by this. Integration with Socket.io allows real-time rendering in multiple windows or devices.
